Le HTML 5

Les sections

Avec le HTML 5 , la nouveauté se porte sur des noms de section( autant le savoir tout de suite !). Ce sont des balises qui viennent structurer la page. Nous retrouvons les anciennes soit: Header/body/footer ainsi que DIV et SPAN. mais nous avons également: NAV, MAIN, ARTICLE SECTION ASIDE

Ces balises peuvent être imbriquées les unes dans les autres. Ainsi, une section peut avoir son propre en-tête.

Ces balises ne s'occupent pas de la mise en page. Elles servent seulement à indiquer à l'ordinateur le sens du texte qu'elles contiennent. On pourrait très bien placer l'en-tête en bas de la page si on le souhaite.

Les Balises

Le HTML est un language informatique qui utilise des balises. C'est à dire des "<" et des ">" contenant des lettres ou des mots. Par exemple '< a>' qui est la balise pour les liens. Voici d'autres exemples:

  • Les Div sont des conteneurs de type bloc. Elles sont utilisée pour strucutrer les pages HTML
  • "h1,2,3,4,5,6":(h pour header) Titres (1 étant le plus important et 6 le moins important
  • "p": paragraphes
  • "Title": permet de mettre un titre a l'onglet. (c'est aussi ce qui est affiché sur les moteurs de recherche!)
  • "en" permet de mettre en italique
  • "strong" : permet de mettre en gras
  • "mark" permet de souligner
  • "UL" permet de commencer une liste à puce
  • "OL" permet de commencer une liste ordonnée
  • "LI" ligne de la liste pour ajouter des lignes à ul ou ol.
  • "a href="nom d'une page/fichier>nom a afficher "" : permet de faire un lien vers la page ou permet de télécharger un fichier. ce dernier doit etre dans le même dossier pour que le téléchargement se lance.

La plupart des balises vont par paire, comme celles vues précédement. En revanche, il en existe également qui qui n'ont pas besoin d'être "fermée". Elles portent plusieurs noms comme par exemple "balises orphelines" ou "balise autofermantes". En voici quelques unes:

  • "br" retour à la ligne
  • "hr/" faire un trait
  • "img" permet d'inserer une image. Doit etre placé dans un paragraphe
  • "video"permet d'insérer une video

Les Attributs.

Les attributs sont commes des adjectifs aux balises. Ils viennent ajouter une information supplémentaire aux balises. L'attribut se place dans la balise ouvrante, ex:

.

Ouverture de balise("<") + nom de la balise + espace + Attribut="complément de l'attribut" +fermeture (">") + affichage + fermeture de la balise ("< /X>")

Voici quelques exemples d'attribut.

  • title="" : permet d'ajouter un texte lorsque le curseur survole le lien
  • src="" : permet de selectionner une image
  • alt="" : permet de décrire une image lors nottement de son insértion
  • id="": permet de "marquer" un endroit afin de permetre un ancrage
  • target="_blank" : permet d'ouvrir un lien dans une nouvelle fenêtre.

Les Classes et les Ids du HTML.

Les classes et les Id permettent au developpeur de classer les balises.

Ce classement est très pratique pour nottament la mise en page avec le CSS. En effet, grâce au CSS, le developpeur peut définir le comportement qu'auront tout les éléments de cette catégorie. Par exemple aligner le texte, changer la couleur etc...

Les classes et les ID sont identique en tout point excepté le fait que l'ID ne peut etre rattaché qu'à un seul élément contrairement donc aux classes qui peuvent être rattachés à de nombreuses balises.

Les Tableaux en HTML

Avant l'arrivé de FLEXBOX, la pratique voulait que les sites internet soient construit avec des tableaux. Ceux ci permettaient de strucuter le site. Maintenant de nombreuses techniques existent afin de pouvoir construire un site internet sans passer par des tableaux. Voici comment créer un tableau:

La balise "table" vient délimiter le tableau.

La balise "Thead" vient délimiter la premiere ligne du tableau

La balise "tr" vient délimiter les lignes du tableau ( première incluse)

La balise "th" cible une cellule

La balise avec l' attribut "td rowspan="2" " permet la fusion d'une colonne sur deux lignes

La balise avec l' attribut "td colspan="2" " permet la fusion d'une colonne sur deux colonnes

Les formulaires

Le HTML permet au developpeur de créer des formulaires. Ceux ci sont devenus incontournables. ex: les retours clients

Les réponses sont stockés sur une autre pages et dans un autre language. Il faudra noter au début d'un formulaire la direction que prendront les informations une fois ceux ci envoyées

Voici quelques notions à connaitre pour la construction d'un formulaire.

Avant de voir les balises des formulaires, il est également important de noter qu'une fois complèter, il faut traiter l'information.

La balise "FORM": permet de délimiter un formulaire.

Il existe 2 attributs pour cette balise: "action" et "method" qui serviront pour le coté back en C#/ php

action = page qui va traiter le formulaire

method = il en existe deux types get ou post

  • get = données dans l'url(limité à 250 caractères)
  • post= données dans la requete ( meilleur en terme de traitement de données)

Champ de renseignement libre

Le balise "Label" sert à écrire à coté du champs du formulaire.

La balise autofermante "Imput type="txt" sert à pouvoir taper du texte dans le champs

Pour lier un label à un imput il faut ajouter un id sur l'imput et un attribut sur le label. Les deux attributs doivent avoir la même valeur pour faire la liaison.

<'.label for="imputFirstname" placehold text="Votre prenom">Votre prenom*:<' /label>
<'input type="text" id="imputFirstname" name="firstname" required />"

Champ de renseignement long

Les balises rows et cols permettent de definir la taille de la zone d'écriture (rows= ligne, cols=colonnes)

<'label for="textareaMessage">Votre message<' /label>
<'textarea id="textareaMessage" name="message" rows="4" cols="40">

Liste déroulante

La balise " select" permet de délimiter la liste déroulante

La balise select prend le role de la balise "imput"

il faut ensuite choisir les choix du menu déroulant:

vous pouvez ajouter l'attribut "value="" " celui ci permet de definir les mots clé a taper pour y accèder plus rapidement (ex "FR" pour France)
vous pouvez également ajouter " selected" pour definir la valeur par défaut

Votre Pays: <'select id="selectcountry" name="country">
